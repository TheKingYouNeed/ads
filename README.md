# Voice Typing Ads (French)

Google Ads playable exports for Voice Typing are in:

- `french/voice-typing/base/index.html`
- `french/voice-typing/boss1/index.html`
- `french/voice-typing/boss2/index.html`
- `french/voice-typing/translation/index.html`

Ready-to-upload ZIP files (one creative per ZIP, `index.html` at root):

- `french/voice-typing/zips/voice_typing_base_fr_google.zip`
- `french/voice-typing/zips/voice_typing_boss1_fr_google.zip`
- `french/voice-typing/zips/voice_typing_boss2_fr_google.zip`
- `french/voice-typing/zips/voice_typing_translation_fr_google.zip`

Notes:

- French copy is intentionally natural and playful (not literal translation).
- Google Ads playable tags are included in each HTML:
  - `meta name="ad.size"`
  - `meta name="ad.orientation"`
  - `exitapi.js` script include
- CTA flow supports Google Exit API via `ExitApi.exit()`.
